UWA Timetable Basic Usage

1. On opening the app you will be presented with the Overview page. Head over to settings where you will find class import settings.

2. It is recommended for first time use that you use the first option ("Read from OLCR (direct method)") to import your classes.

3. Next input your username and password. This is sent over a secure HTTPS connection, and your details are not stored.

4. If everything goes OK, you should get a status message saying a number of classes were imported.

5. Exiting out of the settings and going back to the Overview page, you can now select a day and week to view classes on.
